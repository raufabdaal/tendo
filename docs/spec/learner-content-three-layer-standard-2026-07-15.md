# Learner Content Three-Layer Standard — 2026-07-15

## Purpose

This standard fixes the tension between two valid concerns:

1. Content must be easy enough for primary learners.
2. Content must be full enough for learners to pass school/PLE-style exams.

The solution is not to choose between short notes and full notes.

The solution is a **three-layer learner model**.

---

## The three layers

### Layer 1 — Core NCDC lesson

This is the must-know lesson content based directly on the NCDC subtopic.

It should be:

- short,
- vertical,
- labelled,
- easy to scan,
- accurate,
- age-appropriate.

Example:

```txt
Atlantic Ocean:
West of Africa.

Indian Ocean:
East and south-east of Africa.
```

### Layer 2 — Exam booster

This is high-yield school/exam support that expands the core without overloading the first explanation.

It may include:

- common map facts,
- common traps,
- school-first examples,
- textbook-style facts,
- atlas-supported facts,
- teacher-reviewed expansion.

Example:

```txt
Mozambique Channel:
Separates Madagascar from mainland Africa.

Suez Canal:
Connects Mediterranean Sea to Red Sea.
```

### Layer 3 — Practice / marking-guide style answers

This trains the learner to answer questions.

It should include:

- short-answer questions,
- structured-question prompts,
- marking-guide style answers,
- common mistakes.

Example:

```txt
Q: Name the ocean west of Africa.
A: Atlantic Ocean.
```

---

## Why this works

This model gives both:

```txt
child readability + exam fullness
```

The learner first understands the topic simply, then gets high-yield exam facts, then practises answering.

---

## Source rule

Layer 1 must be NCDC-backed.

Layer 2 can use approved textbooks, atlases, teacher notes or trusted references, but it must be marked as exam booster or reference support when it goes beyond the exact NCDC extract.

Layer 3 should use marking-guide style language, but do not call it “official UNEB marking guide” unless it is actually sourced from UNEB.

Use:

```txt
Marking-guide style answer
```

not:

```txt
UNEB official answer
```

unless official.

---

## Standing decision

Do not make the core lesson too shallow.

Do not make the first screen too heavy.

Use three layers:

```txt
Core → Booster → Practice
```

That is how Tendo will be both child-friendly and exam-useful.
